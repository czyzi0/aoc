# Advent of Code
Solutions for [Advent of Code](https://adventofcode.com/).

To run the solution for year `Y` and day `D` use:
```bash
make Y=${Y} D=${D}
```

| Year | Language |
| ---- | --- |
| 2025 | C |
| 2024 | [Lua](https://www.lua.org/) |
| 2023 | [Python](https://www.python.org/) |
| 2022 | JavaScript |
